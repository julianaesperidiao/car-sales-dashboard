# 🚗 Car Sales Dashboard

Este projeto é uma aplicação web interativa construída com Streamlit para análise exploratória de dados de anúncios de carros.

## 🌐 Live App
https://car-sales-dashboard-x8gx.onrender.com

## 📊 Features
- Histograma interativo de preços de carros
- Gráfico de dispersão mostrando preço versus quilometragem
- Interação do usuário por meio de caixas de seleção
- Layout de painel limpo

## 🛠️ Technologies
- Python
- Pandas
- Plotly Express
- Streamlit

## ▶️ Run locally
```bash
streamlit run app.py
